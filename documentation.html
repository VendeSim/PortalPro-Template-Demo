Introdução

Obrigado por adquirir o template PortalPro. Esta é uma solução front-end completa e autônoma, construída com HTML, Tailwind CSS e JavaScript vanilla para máxima performance e facilidade de customização.

Este template inclui:

  • Dashboard completo com cards de KPI e gráfico interativo.
  • Páginas de Solicitações, Calendário, Conversas e Perfil.
  • Página de Gestão de Documentos com upload via modal.
  • Uma vitrine de Componentes para fácil reuso.
  • Suíte de autenticação com telas de Login e Cadastro.
  • Design totalmente responsivo (desktop, tablet, mobile) e com Modo Escuro.

Primeiros Passos

Este template foi projetado para ser o mais simples possível de usar. Não há necessidade de compilação ou instalação de pacotes. Basta abrir os arquivos .html em seu navegador.

  • index.html: O arquivo principal da aplicação após o login.
  • login.html: A página de Login e Recuperação de Senha.
  • cadastro.html: A página de Cadastro de novos usuários.

Todos os estilos e scripts necessários já estão inclusos e são carregados via CDN (Content Delivery Network) para sua conveniência.

Customização

A customização do template é feita diretamente nos arquivos HTML, principalmente no bloco <script> de configuração do Tailwind e no JavaScript principal.

Cores e Tema

As cores principais do template podem ser alteradas facilmente no objeto de configuração do Tailwind, localizado no <head> de cada arquivo HTML.

tailwind.config = {
    theme: {
        extend: {
            colors: {
                'brand-accent': '#38A89D', // Cor principal de destaque
                // ... outras cores
            },
        }
    }
}

Altere o valor hexadecimal de 'brand-accent' para a cor primária da sua marca e o template se adaptará automaticamente.

Fontes

A fonte padrão é a "Inter", carregada do Google Fonts. Para alterá-la, edite a tag <link> no <head> do arquivo e atualize a propriedade font-family no CSS.

<!-- Mude a URL para a fonte desejada -->
<link href="https://fonts.googleapis.com/css2?family=Sua-Fonte:wght@400;500;600;700&display=swap" rel="stylesheet">

<!-- Atualize o CSS -->
<style>
    body {
        font-family: 'Sua-Fonte', sans-serif;
    }
</style>

Itens do Menu Lateral

Os links da sidebar são gerados dinamicamente a partir de um array de objetos em JavaScript, localizado no arquivo index.html. Para adicionar, remover ou editar um item, basta modificar o array navItems.

const navItems = [
    { id: 'dashboard', label: 'Dashboard', icon: icons.dashboard, hasBadge: false },
    { id: 'solicitacoes', label: 'Solicitações', icon: icons.ferias, hasBadge: false },
    // Adicione um novo item aqui
    { id: 'nova-pagina', label: 'Nova Página', icon: icons.algumIcone, hasBadge: false },
];

Lembre-se de criar o conteúdo correspondente na variável contentSections para que o link funcione.

Dados do Gráfico

Os dados do gráfico no Dashboard também estão em um objeto JavaScript, dentro da função handleDashboardCharts. Você pode alterar as labels (meses) e os dados (valores) facilmente.

const chartData = {
    labels: ['Maio', 'Junho', 'Julho', 'Agosto', 'Setembro'],
    datasets: [{
        label: 'Solicitações',
        data: [8, 12, 19, 15, 22],
        backgroundColor: '#38A89D',
    }]
};
